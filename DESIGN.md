# カードインフレ設計方針

## 基本コンセプト

カードのシナジーとコンボを重要視し、単純なステータスインフレではなく、戦略的なビルド構築を促進する。

## 方針

### 1. AD値の扱い
- AD値自体は**あまり上がりにくい**設計にする
- 固定値の加算ではなく、倍率による乗算を活用する

### 2. ダメージ計算の基本
```
最終ダメージ = (ベースダメージ + AD) × 倍率
```

### 3. コンボ・シナジーの重視
- カードを使うたびに強化される効果
- 特定の条件で発動するボーナス
- 連鎖によるダメージ増幅

## 実装例：征服者の再設計

### 変更前
- ベースダメージに固定+50を加算

### 変更後
```
ダメージ = (ベースダメージ + AD) × 倍率
倍率：使用するたび+30%（累積）
初期倍率：100%
```

**効果例**
- 1回目：(ベース+AD) × 1.0
- 2回目：(ベース+AD) × 1.3
- 3回目：(ベース+AD) × 1.6
- 4回目：(ベース+AD) × 1.9

このように、同じカードを繰り返し使うことで指数的に強くなる設計。
